# Kaggle Competition Explorer

**Быстрый поиск, скачивание и интеллектуальный анализ соревнований Kaggle прямо из Python**

Современное full-stack приложение, которое помогает Kaggle-аналитикам и дата-сайентистам мгновенно находить подходящие соревнования, получать подробные описания от ИИ, скачивать датасеты и получать готовые рекомендации по EDA и feature engineering.

![Streamlit UI](https://via.placeholder.com/800x400/05060f/67e8f9?text=Kaggle+Competition+Explorer)  
*(Скриншот интерфейса — красивые карточки соревнований с hover-эффектами и glassmorphism-дизайном)*

---

## ✨ Основные возможности

- **Умный поиск** соревнований по ключевым словам
- **Красивый современный UI** на Streamlit с карточками и анимациями
- **Автоматическое описание соревнования** от GPT-4o-mini (на русском языке)
- **Однокнопочное скачивание** датасета + распаковка всех архивов
- **Глубокий анализ CSV-файлов**:
  - Статистика (строки, столбцы, пропуски, дубликаты)
  - Типы данных, describe()
  - Пропуски по столбцам
  - **ИИ-рекомендации** по анализу и улучшению скора
- **История скачиваний** в SQLite
- Полная автономность — всё работает **локально** на вашей машине

---

## 🛠 Технологический стек

| Слой              | Технология                          |
|-------------------|-------------------------------------|
| **Frontend**      | Streamlit + Custom CSS (glassmorphism, Inter font, hover-анимации) |
| **Backend**       | FastAPI + Uvicorn                   |
| **Data Science**  | pandas, smart CSV reader            |
| **Kaggle**        | официальная `kaggle` Python API     |
| **ИИ**            | Bothub (GPT-4o-mini)                |
| **База данных**   | SQLite (`downloads.db`)             |
| **Хранение**      | Локальная файловая система (`~/Downloads/kaggle_data/`) |

Установка и запуск
1. Клонируйте репозиторий
git clone <ваш-репозиторий>
cd pet_project

2. Создайте и активируйте виртуальное окружение
python -m venv venv
# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

3. Установите зависимости
pip install -r requirements.txt

4. Настройте API-ключи
config -> config.py # BOTHUB_API_KEY = "your_bothub_api_key_here"

5. Запуск приложения
pip install honcho

📋 Как пользоваться

Поиск — введите ключевые слова (например: tabular, computer vision, nlp).
Просмотр — нажмите «Открыть» на карточке соревнования.
Описание — автоматически генерируется ИИ на русском языке.
Скачивание — нажмите «Скачать датасет и проанализировать».
Анализ — выберите CSV-файл и запустите первичный анализ. Получите метрики + рекомендации ИИ.
